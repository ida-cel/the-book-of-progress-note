# git clean 명령어와 옵션

>git을 사용해 소스 코드를 관리하다 보면 변경 사항과 관련된 여러가지 작업을 해야하는 상황이 생깁니다.

### `git clean`
Git 저장소에서 추적되지 않는 파일들을 삭제하는 명령어입니다. 이 명령어는 작업 트리를 정리하기 위해 사용되며, 현재 디렉토리에서 시작하여 버전 관리 하에 있지 않은 파일들을 재귀적으로 제거합니다.
